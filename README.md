# GreenStride

## Project Overview
GreenStride is a web application designed to calculate carbon emissions for students, promoting awareness and encouraging sustainable practices. The project was developed using HTML, CSS, and JavaScript (Vanilla).

## Features
- User-friendly interface for inputting data
- Real-time calculations of carbon emissions
- Visual representation of carbon footprint

## Technologies Used
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **Data Handling**: Custom algorithms for calculating emissions

## How to Run the Project
1. Clone the repository: `git clone https://github.com/bald-cap/GreenStride.git`
2. Open `index.html` in your preferred web browser.
